ScrollBox CSS

Seguramente alguna vez has visto ese efecto vistoso de como se va desplazando un texto en una caja ocultandose tanto en la parte de arriba como en la parte de abajo. Y la verdad es que es muy sencillo de hacer. Ademas de que resulta bastante atractivo como decoracion para darle un toque mas vivo a nuestra pagina.

¿Que necesitamos saber?

En escensia solo hace falta conocimiento de animaciones en CSS y un poco de flexbox. Sumado a un poco de creatividad tendremos un resultado interesante. Asi que aclarando esa duda. ¡Manos a la obra!.

Creando Nuestro container

Comenzaremos creando la clase del div que sera el que contendrá la lista de palabras que vamos ha animar mas adelante. Usaremos un display flex para centrar el texto y la propiedad mas importante del truco es el overflow.

overflow lo que hace es que si el contenido de un elemento se desborda mas alla de las dimensiones del elemento que lo contiene. En lugar de quedar superpuesto se ocultara. ¿Ya vas pillando el truco ;)?

.scrollbox {
  width: 70%;
  height: 50px;
  display: flex;
  align-items: center;
  justify-content: center;
  font-size: 2rem;
  overflow: hidden;
}

Ahora nos enfocaremos en la clase que sera animada: el ul. Como el overflow oculta el contenido desbordante del ul lo único que deberemos hacer es variar el margen superior. Y este como por arte de magia comenzara a desplazarse dentro del contenedor. ¡Y listo! tenemos un bonito efecto. Para que veamos en acción esta idea solo debemos animarla con la propiedad animation.

.scrollbox ul {
  animation: scroll 5s .5s ease-in-out infinite alternate;
}
.scrollbox li {
  list-style: none;
  color: white;
  font-family:'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif;
  text-shadow: 2px 2px rgba(0, 0, 0, .4);
  font-size: 3rem;
}

@keyframes scroll{
  0%{
    margin-top: 136px;
  }
  50%{
    margin-top: 30px;
  }
  100%{
    margin-top: -85px;
  }
}
